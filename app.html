3. Segurança e infraestrutura

            
Como garantimos que seus dados estão seguros

          

        

        

          

            
🏗️

            

              Infraestrutura Supabase (PostgreSQL) — banco de dados hospedado na AWS com certificações SOC2 Type 2, ISO 27001 e GDPR compliance. Nós não operamos servidores próprios — seus dados ficam em datacenters enterprise de classe mundial.
            

          

          
Proteções implementadas

          
    
            
  • Criptografia em trânsito: toda comunicação usa TLS 1.3 (HTTPS)
    • 
            
  • Criptografia em repouso: banco de dados criptografado por padrão pelo Supabase/AWS
    • 
            
  • Row Level Security (RLS): cada usuário acessa somente os dados da sua família — isolamento no nível do banco de dados
    • 
            
  • Autenticação segura: senhas nunca são armazenadas em texto puro — apenas hash bcrypt via Supabase Auth
    • 
            
  • Sessões com expiração: tokens JWT com tempo de vida limitado e renovação automática
    • 
            
  • Backups automáticos: o Supabase realiza backups point-in-time para recuperação de desastres
    • 
          

          
Row Level Security — o isolamento que protege seus dados

          
Todo acesso ao banco de dados é filtrado automaticamente pelo family_id do usuário autenticado. Isso significa que mesmo que um bug existisse no código da aplicação, seria impossível um usuário acessar dados de outra família — o banco de dados rejeita a consulta.

          

            
✅

            
Sem acesso admin irrestrito. Nem mesmo o desenvolvedor do app consegue consultar seus dados financeiros sem autenticar como você. O RLS é aplicado a todos, inclusive ao service_role em operações de leitura.

          

          
O que você pode fazer para se proteger

          
    
            
  • Use uma senha forte e única (mín. 12 caracteres, com letras, números e símbolos)
    • 
            
  • Não compartilhe suas credenciais com ninguém
    • 
            
  • Acesse o app somente em redes confiáveis
    • 
            
  • Revise regularmente os membros da sua família no app
    • 
            
  • Altere sua senha se suspeitar de acesso não autorizado
    •

Como usamos seus dados

  • Armazenar seus dados financeiros de forma segura no Supabase para que você possa acessá-los de qualquer dispositivo
    • 
          
  • Usar dados de telemetria anonimizados para identificar erros técnicos e melhorar o app

🚫 O que NÃO fazemos

        
    
          
  • Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais
    • 
          
  • Não acessamos seus dados financeiros sem sua autorização explícita
    • 
          
  • Não exibimos anúncios com base nos seus dados financeiros
    • 
          
  • Não armazenamos senhas em texto puro — são processadas com bcrypt pelo Supabase Auth
    • 
          
  • Não acessamos sua conta bancária real — todos os dados são inseridos manualmente por você

Retenção de dados e cookies

        
Retenção de dados

        
Seus dados são mantidos enquanto sua conta estiver ativa. Após solicitar a exclusão, os dados são removidos em até 30 dias. Dados de telemetria são retidos por 90 dias para fins técnicos.

        
Armazenamento local (localStorage)

        
O app usa localStorage no seu navegador para armazenar cache temporário, preferências de interface e chaves de API que você configurar. Esses dados ficam somente no seu dispositivo e podem ser limpos em Configurações → Cache ou limpando os dados do navegador.

        
Service Workers (PWA)

        
Para funcionamento offline, o app registra um service worker que faz cache dos arquivos estáticos (código, CSS). Nenhum dado financeiro é armazenado no cache do service worker.

Política de Privacidade

        
Seu controle sobre seus dados financeiros. Transparência total sobre como coletamos, usamos e protegemos suas informações.

        
📅 Última atualização: Janeiro 2025

      


      
      

        

          
🌍

          Conformidade Legal
        

        

          
🇧🇷 LGPD Compliant

          
🇪🇺 GDPR Ready

          
🏆 SOC 2 (Supabase)

          
🔐 TLS 1.3

          
🛡️ ISO 27001

        

        
✅ O Family FinTrack opera em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018) e com as diretrizes do GDPR europeu.

      


      
      

        

          
📋

          Dados que coletamos
        

        
Dados de cadastro: Nome, e-mail (necessário para login e recuperação de senha)

        
Dados financeiros: Transações, contas, categorias, orçamentos e metas inseridos por você

        
Dados de uso (telemetria): Páginas acessadas, tipo de dispositivo e navegador — para melhorar o app (anônimos e agregados)

        
Dados de notificação: Número de WhatsApp e Chat ID do Telegram, apenas se você configurar voluntariamente

        
🚫 Nunca coletamos: dados de cartão de crédito, senhas bancárias, CPF/CNPJ ou qualquer dado sensível não necessário ao funcionamento do app.

      


      
      

        

          
⚙️

          Como usamos seus dados
        

        
Prover todas as funcionalidades do app (lançamentos, relatórios, orçamentos, etc.)

        
Enviar notificações de transações programadas (somente se configurado por você)

        
Melhorar o desempenho e a experiência do app com base em dados anônimos de uso

        
Gerar insights financeiros com IA (Gemini) — os dados são enviados à API do Google de forma segura e não são armazenados pelo Google para treinamento

        
🚫 Nunca vendemos seus dados para terceiros. Nunca usamos seus dados financeiros para publicidade.

      


      
      

        

          
🏦

          Onde seus dados ficam armazenados
        

        
Supabase (PostgreSQL): Banco de dados cloud com certificação SOC 2 Type II e conformidade com GDPR. Servidores na região configurada pelo administrador.

        
Row Level Security (RLS): Políticas de segurança em nível de banco de dados garantem que cada família acessa apenas seus próprios dados — tecnicamente impossível acessar dados de outra família.

        
Backup automático: O Supabase mantém backups automáticos por 7 dias. Você pode exportar seus dados a qualquer momento em CSV ou PDF.

        
Dados locais: Algumas preferências são armazenadas no seu dispositivo (localStorage) e nunca saem dele.

      


      
      

        

          
🔗

          Serviços de terceiros utilizados
        

        
Supabase — Banco de dados e autenticação. Política de privacidade ↗

        
Google Gemini AI — Análise de recibos e insights financeiros (apenas quando ativado). Política ↗

        
EmailJS — Envio de notificações por e-mail (apenas quando configurado pelo admin). Política ↗

        
Nenhuma rede de publicidade — O app não contém rastreadores de anúncios, cookies de terceiros para publicidade ou qualquer serviço de analytics invasivo.

      


      
      

        

          
⚖️

          Seus direitos (LGPD / GDPR)
        

        
Acesso: Visualize todos os seus dados nos Relatórios e Exportação do app

        
Retificação: Edite qualquer informação diretamente no app

Apagamento: Solicite a exclusão completa de todos os dados abaixo

        
Oposição: Desative a telemetria nas Configurações → Família (se disponível)
