      🤖 Leitura de Recibos com IA
      
    

    

      

        O app usa o Google Gemini 2.0 Flash para ler fotos de recibos,
        notas fiscais e PDFs e preencher automaticamente os campos da transação.
        A chave fica salva no banco, acessível apenas pela sua família.
      


      
      

        Como obter sua chave (gratuita):

        1. Acesse aistudio.google.com/app/apikey

        2. Clique em Create API Key

        3. Cole a chave abaixo (começa com AIza)

        ✓ Camada gratuita: 1.500 leituras/dia sem custo
      


      

        
        

          
          
        

      

    

    

      
      
    

  






  

    

      📧 Configurar EmailJS
      
    

    


      
      

        
        Não configurado
        
      


      
      

        
        

          
          
        

        
Em Email Services → clique no serviço → Service ID

      


      

        
        

          
          
        

        
Em Email Templates → clique no template → Template ID

      



      

        
        

          
          
        

        

          Se vazio, usa o Template ID padrão. Use este template para notificações de transações programadas (auto-register / lembretes).
        

      



      

        
        

          
          
        

        
Em Account → General → Public Key

      


      


      
      

        

          ⚠️ Passo obrigatório no painel do EmailJS:

          No seu template, o campo "To Email" deve ser definido como {{to_email}} — caso contrário o envio falhará com erro "recipients address is empty".

          Abrir Templates no EmailJS ↗
        

        📋 Variáveis disponíveis no template EmailJS:


        Roteamento

        {{to_email}} destinatário  · 
        {{from_name}} remetente  · 
        {{subject}} assunto


        Corpo do e-mail

        {{message}} mensagem do usuário  · 
        {{report_period}} período  · 
        {{report_view}} tipo de relatório

        {{report_income}} receitas  · 
        {{report_expense}} despesas  · 
        {{report_balance}} saldo  · 
        {{report_count}} nº de transações


        📎 Link do PDF (Supabase Storage)

        {{pdf_url}} URL pública de download do relatório

        {{pdf_name}} nome do arquivo PDF gerado

        O PDF é salvo automaticamente no bucket fintrack-attachments/reports/ com visibilidade pública.
