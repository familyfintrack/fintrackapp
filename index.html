O Family FinTrack é um Progressive Web App (PWA) — uma categoria especial de aplicativo que combina o melhor da web com a experiência de um app nativo. Você não precisa passar pela App Store ou Google Play para usá-lo.

Basta acessar pelo navegador com conexão à internet. Ou, se preferir, instale com um clique na tela inicial — e ele se comportará como qualquer outro app: com ícone próprio, tela cheia e acesso rápido.

Seus dados são seus.
Somente seus.

Construímos o Family FinTrack com uma premissa inegociável: nenhum dado financeiro da sua família pode ser acessado, vendido ou visualizado por ninguém — nem mesmo por nós.

Zero Access
Nem nós vemos
🔐
Criptografia de ponta
Senhas protegidas com PBKDF2 + salt único por usuário. Dados em trânsito via TLS 1.3. Dados em repouso com AES-256. Nenhuma senha é armazenada — apenas seu hash irreversível.
✓ PBKDF2 + Salt ✓ TLS 1.3 ✓ AES-256
🏰
Isolamento total por família
Row-Level Security no banco de dados garante — ao nível do servidor de banco — que a família A nunca acessa dados da família B. Isso não é regra de negócio: é uma trava no banco de dados que não pode ser contornada.
✓ RLS PostgreSQL ✓ JWT isolado ✓ Multi-tenant
👁️‍🗨️
Zero acesso nosso
Não temos acesso ao conteúdo das suas transações, valores ou categorias. Dados de telemetria coletam apenas navegação anônima (páginas visitadas, erros técnicos) — sem nenhuma informação financeira.
✓ Sem acesso financeiro ✓ Telemetria anônima
Garantias técnicas, não promessas de marketing
🗄️
Banco de dados isolado
PostgreSQL com RLS ativo em cada tabela. Uma família = um escopo. Verificável e auditável.
🔑
Chaves que não viajam
Credenciais de API e chaves sensíveis armazenadas em vault criptografado separado do banco principal.
📵
Sem anúncios. Jamais.
Seus dados financeiros nunca são vendidos, compartilhados ou usados para segmentação publicitária. Nunca.
🇧🇷
LGPD compliant
Direito de acesso, portabilidade e exclusão total dos seus dados a qualquer momento, conforme a Lei Geral de Proteção de Dados.
🧹
Exclusão permanente
Ao solicitar exclusão, seus dados são removidos em cascata de todas as tabelas em até 30 dias. Sem rastros.
🔒
Modo privacidade no app
Oculte todos os valores com um clique. Ideal para usar o app em locais públicos sem expor sua situação financeira.
